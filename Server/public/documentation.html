Documentation


This is the documentation for the website, F/stop API. A Movie Directory API focussed on film pieces.


Endpoints


  • '/': Returns a welcome message via GET

  • '/movies': Returns a list of all movies on F/stop's API via GET

  • '/movies/:Title': Returns data (description, genre, director, image URL, whether its featured or not) about a single movie by title to the use via GET

  • '/genres/:Name': Returns data about a genre (description) by name/title (e.g., “Thriller”) via GET

  • '/movies/:Genre': Finds all movies of a given genre via GET

  • '/directors/:Name': Returns data about a director (bio, birth year, death year) by name via GET

  • '/movies/:Director': Returns data about a directors work/movies they have done via GET

  • '/users': Allows new users to register via POST

  • '/users/:Username': Allows users to update their user info (e.g. username) via PUT

  • '/users/:Username/movies/:MovieID': Allows users to add a movie to their list of favorites via POST

  • '/users/:id/movies/:movieTitle': Allow users to remove a movie from their list of favorites via DELETE

  • '/users/:id': Allows existing users to deregister via DELETE
