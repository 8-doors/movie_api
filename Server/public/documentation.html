Documentation


This is the documentation for the website, F/stop API. A Movie Directory API focussed on film pieces.


Endpoints


  • '/': Returns a welcome message via GET

  • '/Movies': Returns a list of all movies on F/stop's API via GET

  • '/Movies/:movieTitle': Returns data (description, genre, director, image URL, whether its featured or not) about a single movie by title to the use via GET

  • '/Movies/Genre/:genreName': Returns data about a genre (description) by name/title (e.g., “Thriller”) via GET

  • '/Movies/Director/:directorName': Returns data about a director (bio, birth year, death year) by name via GET

  • '/Directors/:directorName': Returns data about a directors work/movies they have done via GET

  • '/Users': Allows new users to register via POST

  • '/Users/:_id': Allows users to update their user info (username) via PUT

  • '/Users/:_id/Favorites': Allows users to add a movie to their list of favorites via POST

  • '/Users/:_id/Favorites/:movieTitle': Allow users to remove a movie from their list of favorites via DELETE

  • '/Users/:_id': Allows existing users to deregister via DELETE
